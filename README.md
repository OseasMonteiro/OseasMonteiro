## 🧮 Por que um Professor de Matemática vira Cientista de Dados?

> *"Ensinar estatística para adolescentes me ensinou que dado sem contexto é ruído. A Ciência de Dados me dá as ferramentas para transformar esse ruído em conhecimento."*

A docência me deu habilidades que poucos cientistas de dados têm:
- **Comunicação clara** de conceitos complexos para diferentes públicos
- **Pensamento lógico** aplicado à resolução de problemas reais
- **Paciência e metodologia** para ensinar algoritmos, que agora aplico para depurar modelos
- **Visão de impacto** — sempre foco no que os dados significam para as pessoas
